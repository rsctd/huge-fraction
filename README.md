# huge-fraction
A PHP library to work with huge fractions

### Prerequisites
 - PHP version >= 7.0.0

### Installation
In your project folder:

```shell
composer require rsctd/huge-fraction
```

### Usage
```php
use Rsctd\HugeFraction\Fraction;
```

### Running tests
To run the tests locally, run the command
```
composer test
```
